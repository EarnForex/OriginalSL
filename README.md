# OriginalSL

OriginalSL is a custom indicator for MT5 and cTrader by EarnForex.com.

OriginalSL displays the original stop-loss order values for your historical trades.

Use it to analyze how your initial SL would fare against the trailing/BE/modified one.

![OriginalSL - an example of the hourly EUR/USD chart with the original SL marked for two buy trades](https://github.com/EarnForex/OriginalSL/blob/main/originalsl-example-long-eurusd-trade.png)

A detailed description of the indicator is available here:
https://www.earnforex.com/indicators/OriginalSL/
